# Changelog

All notable changes to this mod will be documented in this file.

## [1.3.0] - 2026-03-14

### Added
- Arc Pistol: Chain lightning pistol (闪电戟)
  - Neolithic tech level (初期可用)
  - 25 射程，30 伤害，1.5s 充能
  - Combat Extended 支持
  - 需要 5 钢 + 1 组件制作

## [1.2.0] - 2026-03-12

### Added
- Flying Tigers Elite Helmet: Elite head protection (200 Sharp/Blunt, 3 Electric armor)
- Flying Tigers Elite Armor: Elite combat armor (200 Sharp/Blunt, 33 Electric armor)

### Changed
- Helmet and Armor attributes balanced to 20% of original LWOP values

## [1.1.2] - 2026-03-12

### Fixed
- Fixed ammo recipe not showing on loading bench (use CE AmmoRecipeBase)
- Added workAmount to recipes
- Added missing recipeMaker properties for Joywire
- Reduced Super Rifle cost to 1 steel

## [1.1.1] - 2026-03-12

### Added
- Super Barrett Ammo: Overpowered anti-materiel ammunition for the Super Rifle
- Ammo crafting recipe (1 steel = 1000 rounds)

### Fixed
- Fixed ammo not loading into weapon (changed defaultProjectile)
- Added missing ammo description and ammoClass
- Fixed bullet texture path
- Added missing projectile properties (damageDef, dropsCasings)

## [1.0.0] - 2026-03-11

### Added
- Joywire implant (permanent mood boost)
- Super Rifle (CE compatible)
- Mod icon and preview images
- GitHub repository

### Fixed
- Combat Extended packageId

### Changed
- Updated mod description with LWOP credit
