# Changelog

All notable changes to this mod will be documented in this file.

## [1.3.0] - 2026-03-22

### Changed
- Split mod into CE and Vanilla versions
- CE version: requires Combat Extended, uses Bullet_ChargeRifle
- Vanilla version: standalone, no CE dependency, uses native projectile
- Updated all stat fields to match LWOP best practices
- Armor and helmet: add JumpRange, PackRadius, PsychicEntropyGain, full Royalty stats
- Helmet: add VacuumResistance=100, ToxicEnvironmentResistance=99, Royalty entropy stats

## [1.2.1] - 2026-03-17

### Added
- HyperVitality Chip: Neural implant that reduces sleep need to 2 hours per day

### Changed
- Adjusted HyperVitality Chip restFallFactor from -2 to -0.75 (2h sleep)
- Added ArmorRating_Heat:10 to Flying Tigers helmet and armor (thermal protection)

## [1.2.0] - 2026-03-12

### Added
- Flying Tigers Elite Helmet: Elite head protection (200 Sharp/Blunt)
- Flying Tigers Elite Armor: Elite combat armor (200 Sharp/Blunt)

### Changed
- Helmet and Armor attributes balanced

## [1.1.2] - 2026-03-12

### Fixed
- Fixed ammo recipe not showing on loading bench
- Added workAmount to recipes
- Reduced Super Rifle cost to 1 steel

## [1.1.1] - 2026-03-12

### Added
- Super Barrett Ammo: Overpowered anti-materiel ammunition
- Ammo crafting recipe

### Fixed
- Fixed ammo not loading into weapon
- Fixed bullet texture path

## [1.0.0] - 2026-03-11

### Added
